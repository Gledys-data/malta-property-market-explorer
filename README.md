# Project Overview

The **Malta Property Market Explorer** is a hands-on data analytics project that documents my journey learning **Python**, **web scraping**, **data analysis**, and **automation** using real estate listings from the Malta property market.

Rather than building everything at once, this project is developed step by step through a series of Jupyter notebooks. Each notebook introduces a new concept while building on the knowledge gained in the previous one.

This project has two main purposes:

* To develop practical Python and data analytics skills by working with real-world data.
* To build a dataset that will help me better understand the Malta property market as I prepare to purchase a property in the future.

This repository documents my learning journey as I progress from the fundamentals of web scraping to building an end-to-end property market analysis project using real-world data.

# How to Run This Project

## 1. Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/malta-property-market-explorer.git
```

Replace `YOUR_USERNAME` with your GitHub username.

## 2. Navigate to the project folder

```bash
cd malta-property-market-explorer
```

## 3. Install the required libraries

```bash
pip install -r requirements.txt
```

## 4. Launch Jupyter Notebook

```bash
jupyter notebook
```

Open the `notebooks` folder and run the notebooks in numerical order.
