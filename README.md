# Malta Property Market Explorer

## Project Objective

This project explores property prices in Malta using Python, web scraping, and data analysis.

The goal is to practice data collection, cleaning, and analysis while creating a useful tool to understand the Malta housing market.

## Tools Used

- Python
- BeautifulSoup
- Requests
- Pandas
- Matplotlib
- Jupyter Notebook

## Data Ethics

This project is for educational purposes only.

The project does not publish:
- personal contact details
- agent names
- phone numbers
- emails
- exact addresses
- property photos
- full listing descriptions

Only cleaned and aggregated data will be considered for publication.
